BLOP is a plotting utility with the main goal to create figures to be included in LaTeX documents. Text in the plots is typeset by LaTeX (which is currently the best quality typesetter for mathematical texts, which I know of). The plots can be scaled in the horizontal and vertical directions independently, with no distortions. The fontsize/fontstyle in the plots is the same as in the surrounding LaTeX document, regardless of the scaling of the plot. The best start to get an impression is definitely looking at the example section of this documentation.
