Drawing horizontal/vertical lines with optional labels

The utility functions to add horizontal/vertical marker lines are defined in the namespace set. 
line &set::aline(double d, int ax)
This draws and returns a marker line at the given value of the specified axis. For example, set::aline(1,axis::y2); draws a horizontal marker line extending the full width of the given frame, at the value 'd' of the y2 axis. 
line &set::aline(double d, int ax, var label, const length &x=-0.5*EX, const length &y=0.05, sym::position halign=sym::right, sym::position valign=sym::bottom, double angle=unset);
This draws a marker line as before, and adds a label to it. x and y are absolute or relative positions from the line position, depending on whether it's a horizontal or vertical line. The other parameters are self-explanatory.

The following shorthands can also be used: 

line &vline(double d);
line &v1line(double d);
line &v2line(double d);
struct{line,label} vline(double d,var label, const length &dx=-0.5*EX, const length &y=0.05, sym::position halign=sym::right, sym::position valign=sym::bottom, double angle=90*unit::deg)
Draw a vertical line at the given value of the x1, x1 and x2 axes, respectively. The last one adds a label as well. The same functions (v replaced by h) exist for horizontal lines. 
vline(double d, var label, const length &dx=-0.5*EX, const length &y=0.05, sym::position halign=sym::right, sym::position valign=sym::bottom, double a=90*unit::deg);
This draws a vertical line at the value 'd' of the y1 axis, and adds a label to it. It returns a structure { line the_line; label the_label; }; which can be captured with this syntax: 
auto [lin,lab] = vline(1,"Hello");
The same function exists for horizontal line. 
aline(double d, int ax, var label, const length &dx=-0.5*EX, const length &y=0.05, sym::position halign=sym::right, sym::position valign=sym::bottom, double a=90*unit::deg);
Same as before, but you can specify the axis (axis::x1, axis::x2, axis::y1, axis::y2).
