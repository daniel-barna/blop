Sorting data
Practical Guide: 
// plot a data, sorted, from a file, which is unsorted (or not sorted in the way it is plotted)
// Plot 3rd column as the function of the product of the first two columns. Sort the points by this product
// (i.e. _1 in the argument of sort(...) refers to the data graph (which contains the product of the first
// two columns of the file) in its first value
plot("unsorted-data.dat",_1*_2,_3).sort(_1).ds(lines());
The member function sort(const function &) of the dgraph class sorts the data points as a function of the given expression. Since actual plotting and export into whichever output file format occurs when for example pdf::print("filename.pdf"); is executed, calling sort "after" the plot command still has an effect.
