Kinematics (physics)
Practical Guide: 
// Get the momentum of a 100 keV proton
double momentum = kin(938*unit::MeVc2).Ekin(100*unit::keV).p();
blop::phys::kin is a utility class to calculate kinematic variables (momentum, energy, kinematic energy, velocity) from each other, given the particle's mass. See example above.
